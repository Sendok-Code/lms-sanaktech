TEST 1: Buka Console (F12)
--------------------------
1. Tekan F12
2. Klik tab "Console"
3. Lihat ada ERROR MERAH?
4. Screenshot dan kirim

TEST 2: Check Button Element
----------------------------
Di Console, ketik ini dan Enter:

    document.getElementById('complete-btn')

Harus return: <button ...>

Jika return null, berarti button tidak ada.

TEST 3: Check Function
---------------------
Di Console, ketik ini dan Enter:

    typeof markAsComplete

Harus return: "function"

Jika return "undefined", berarti function tidak loaded.

TEST 4: Manual Call
------------------
Di Console, ketik ini dan Enter (GANTI 1 dengan ID video saat ini):

    markAsComplete(1)

Lihat apa yang terjadi.

TEST 5: Check Network
--------------------
1. Buka tab "Network" di DevTools (F12)
2. Klik "Tandai Selesai"
3. Lihat ada request ke /student/progress/... ?
4. Klik request tersebut
5. Lihat Status Code (harus 200)
6. Lihat Response (harus {success: true, ...})
7. Screenshot
